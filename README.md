# Rockman-Style Action Game Logic Prototype

Unityを使用した2Dアクションゲームの挙動・ロジックを重視したプロトタイププロジェクトです。
「マップデザイン」よりも、**「キャラクターの操作性や物理的な反応をロジックで制御すること」**に重点を置いて制作しました。

## 🚀 プロジェクトの重点ポイント

本プロジェクトでは、市販のアセットを組み合わせつつ、以下の挙動を自作のC#スクリプトで制御しています。特に「手触りの良さ」を数学的な計算やフラグ管理で実現することに注力しました。

### 1. プレイヤーの高度な挙動制御
- **可変ジャンプ・ダッシュ**: 一定距離を高速移動するダッシュ処理や、空中でのステート管理を実装。
- **射撃システム**: 連射速度（Fire Rate）の調整機能を備えたビーム発射ロジック。

### 2. 敵AIと物理レスポンス
- **自律移動**: 一定周期で左右をパトロールするルーチン。
- **ノックバック処理**: 攻撃を受けた際、ヒットしたベクトルから衝撃の方向を算出し、その方向へ敵を弾き飛ばす「よろけ」の挙動を実装。

### 3. カメラシステム
- **スムース追従**: Lerp（線形補間）を用い、プレイヤーの動きに対してカメラが遅れて滑らかに追従するよう調整。
